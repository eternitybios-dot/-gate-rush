# GATE RUSH 3D

スマホブラウザ向け・縦画面カジュアル3Dランゲーム。1ファイル(`index.html`)完結、外部ライブラリ・画像・音声素材なし。

**遊ぶ:** https://eternitybios-dot.github.io/-gate-rush/

## 遊び方

- 指(またはマウス)で左右にドラッグして部隊を移動
- 部隊は自動で前進しながら自動射撃
- 数字ゲートを選んで兵士を増やす(×2, ×3, +N)
- **赤いゲートは撃ち続けると数字が上がり、青いゲートに変わる**
- 迫りくる変異体の群れを撃ち倒し、ステージ最後の巨大変異体を撃破せよ

## 技術

- Canvas 2D 上に自前実装したパースペクティブ投影による疑似3D描画
- スプライトの事前ベイクによる描画高速化(devicePixelRatio対応)
- Web Audio API によるプロシージャル効果音(素材ファイルなし)
- Vibration API 対応(Android)
- localStorage にベストスコア保存
